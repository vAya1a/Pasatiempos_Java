# Juegos de consola en Java

Proyecto personal realizado en **Java** que reúne varios pasatiempos clásicos jugables desde la **consola**.  
El objetivo principal es practicar lógica de programación y conceptos básicos del lenguaje.

## Juegos incluidos

- **Número Oculto**  
  Juego de adivinanza en el que hay que acertar un número aleatorio con pistas de mayor o menor.

- **Ahorcado**  
  Versión clásica del ahorcado, donde se debe adivinar una palabra letra a letra antes de quedarse sin vidas.

- **Adivinanzas**  
  Juego de adivinanzas basado en pistas progresivas sobre lenguajes de programación.

- **Blackjack**  
  Implementación sencilla del juego de cartas Blackjack contra la banca.

## Notas

- Proyecto desarrollado con fines **educativos y de práctica**.
- El código es mejorable y no pretende ser una solución avanzada o definitiva.
- Todos los juegos se ejecutan en **consola**, sin interfaz gráfica.
